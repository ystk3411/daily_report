## 分かったこと
・ローカルとリモートのリポジトリを合わせるときは
　```
 git pull リモート名 ブランチ名 --rebase
　```
 でできる。

・ローカルとリモートのリポジトリを合わせるときにローカルの方にリモートリポジトリに反映していない変更があった際は`stash`で一時退避。
